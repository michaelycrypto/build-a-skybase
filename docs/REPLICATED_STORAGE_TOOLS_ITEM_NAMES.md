# Complete List of Item Names for ReplicatedStorage.Assets.Tools

This document lists **ALL** item names that should exist as 3D models in `game.ReplicatedStorage.Assets.Tools`.

Models can be named using either:
- **Item name** (e.g., "Apple", "Stick")
- **Item ID** (e.g., "37" for Apple)

The system will look in: `ReplicatedStorage.Assets.Tools[modelName]` (primary) or `ReplicatedStorage.Tools[modelName]` (legacy fallback)

## Summary

**Total: 206 item names** that should exist in `ReplicatedStorage.Assets.Tools`

### Breakdown:
- **Tools**: 5-7 items (required)
- **Food Items**: 5 items (currently implemented)
- **Crafting Materials**: 10 items
- **Seeds & Farming**: 3 items
- **Saplings**: 6 items
- **Decorative**: 2 items
- **Full Blocks**: ~173 items

### Priority:
1. **Critical**: Tools (5 items) - Required for gameplay
2. **High**: Food items (5), Crafting materials (10), Seeds (3)
3. **Medium**: Saplings (6), Ores (7), Ingot blocks (7)
4. **Low**: All full blocks (can use procedural rendering as fallback)

---

## Notes

- Item names should match the **display names** from BlockRegistry (e.g., "Oak Log", "Stone Bricks")
- Spaces in names are preserved (e.g., "Wheat Seeds", "Light Blue Stained Glass")
- The system will fall back to procedural rendering if a model is not found (except for tools)
- Models can be `MeshPart` directly or a `Model`/`Folder` containing a `MeshPart`
